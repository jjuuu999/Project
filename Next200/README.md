# Next200

`Next200`은 KOSPI200 편입/편출 후보를 예측하는 프로젝트입니다.

이 저장소는 아래 두 버전으로 정리되어 있습니다.

## 버전 구성

### v1

팀 프로젝트 기반의 초기 버전입니다.
기본 파이프라인, 예측 로직, Streamlit 앱 구성이 포함되어 있습니다.

- [Next200_v1](./Next200_v1/)

### v1.5

`v1`을 바탕으로 모델과 피처 조합을 다시 실험해 성능을 개선한 개인 확장 버전입니다.
최종 모델, 실험 노트북, 성능 시각화, 실행 가이드는 이 버전에 정리되어 있습니다.

- [Next200_v1.5](./Next200_v1.5/)

### v2

다음 버전은 현재 개발 중입니다.
